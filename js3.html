var nickname = "ichiro"

let nickname = "taro"

let nickname = "ichiro"
console.log(nickname)

const nickname = "taro"

onsole.log(nickname)

関数

「関数（function）」は、繰り返し使われる一連の処理を1つにまとめたもの
最も簡単な関数は以下のように記述
function 関数名(){
処理内容
}

「引数」と「戻り値」

入力する値を「引数（ひきすう）」、出力される値を「戻り値（もどりち）」
引数、戻り値は以下のように書く

function 関数名(引数){
処理
return 戻り値;
}
