JavaScriptでの変数の書き方

let hello = 'Hello World';
alert(hello);
これでアラートにHello Worldが表示される

四則演算の書き方

足し算
alert(4 + 3);
引き算
alert(10 - 3);
掛け算
alert(2 * 3);
割り算
alert(21 / 3);

文字列を結合する

alert('Hello' + 'World')
これにより、アラートで'HelloWorld'が表示される

また、こういった書き方がある
let str1 = 'Hello';
let str2 = 'World';
alert(str1 + str2);
アラートで'HelloWorld'と表示される。

条件分岐

条件式が一つの場合

if(条件1){
処理1
}
上記は条件1を満たすときは、処理1を実行

1つの条件式で、処理を2つに分ける場合

if(条件1){
処理1
} else{
処理2
}
上記は条件1を満たすときは処理1、満たさないときは処理2を実行

2つの条件式で処理を分ける場合

if(条件1){
処理1
} else if(条件2){
処理2
} else{
処理3
}
上記は条件1を満たすときは処理1、条件2を満たすときは処理2、どちらも満たさないときは処理3を実行

[例]
let orange = 100;
let apple = 120;
if(orange < apple){
alert('みかんの値段がりんごより安い');
} else if(orange == apple){
alert('みかんとりんごが同じ値段');
} else{
alert('みかんの値段がりんごより高い');
}
値段の変動で、アラートで表示される処理内容が変わる。

while文

「while文」では、条件式が真（true）である間は繰り返し処理が実行されます。
while(条件){
処理
}
[例]
let max = 100;
let num = 1;
let count = 0;
while(num < max){
num = num * 2;
count = count + 1;
}
alert('2を掛け続けて' + max + 'を超えるのに必要だった回数は' + count + '回です');
numに掛け続ける数の変動でアラートで表示される処理内容が変わる。

do...while文

条件式のtrue／falseに関わらず、最初の1回だけは必ず処理が行われる。
その後、条件式を評価して、繰り返し処理を続けるか／抜けるかを決めていきます
do{
最低1回は行われる処理
}while(条件式);

for文

決められた回数の処理を繰り返す
for ( 初期値; 条件式; 増減値 ){
繰り返し処理
}
[例]
let i; :「変数i」を宣言
let num = 0; :「変数num」を宣言し、初期値を「0」に設定
for(i = 1; i < 11; i++){
:変数iの初期値を「1」に設定し、変数iが「11」未満のときは変数iに1を足す
（「i++」は「i = i+1」と同じ）
num = num + i; :変数numに「num+i」を代入
}
alert('1から10まで足し算した結果は' + num + 'です');
この場合のアラート結果は「1から10まで足し算した結果は55です」となる
